# ⚡ Demand Smart

Sistema de gestão inteligente para monitoramento de pivôs de irrigação, manutenção, consumo energético e controle financeiro operacional.

---

## 📌 Visão Geral

O **Demand Smart** é uma plataforma voltada para gestão agrícola/industrial, permitindo o controle centralizado de:

- 💧 Pivôs de irrigação  
- 🔧 Manutenções preventivas e corretivas  
- ⚡ Consumo e custo energético  
- 💰 Gestão financeira operacional  
- 📊 Dashboards e indicadores inteligentes  

---

## 🚀 Funcionalidades

### 💧 Pivôs
- Cadastro de pivôs
- Monitoramento de status
- Controle de operação

### 🔧 Manutenção
- Registro de manutenções
- Tipos: preventiva, corretiva e emergencial
- Controle de custos por equipamento

### ⚡ Energia
- Monitoramento de consumo em tempo real
- Estimativa de custo energético
- Análise de eficiência

### 💰 Financeiro
- Consolidação de custos:
  - Manutenção
  - Energia
  - Aquisição de pivôs
- Visão de custo total operacional
- Relatórios financeiros

### 📊 Dashboard
- KPIs principais
- Gráficos de desempenho
- Indicadores de eficiência
